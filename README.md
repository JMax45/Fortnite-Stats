Fortnite Stats is a Python program for see a fortnite player stats

## Installation

### Requirements
* Python 3.3 and up

`$ pip install fortnite-python`
`$ pip install Pillow`

`Get an API key on: https://fortnitetracker.com/site-api`
`Input your API key on line 22`
